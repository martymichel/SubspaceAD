SubspaceAD: Training-Free Few-Shot Anomaly Detection
via Subspace Modeling

CVPR 2026

arXiv Code 🤗 Demo

Overview

SubspaceAD is a training-free method for few-shot industrial anomaly detection. It extracts patch-level features from a small set of anomaly-free images using a frozen DINOv2 backbone and fits a PCA model to estimate a low-dimensional subspace that captures normal appearance variation. At inference, anomalies are detected and localized via reconstruction residuals with respect to this subspace, yielding interpretable and statistically grounded anomaly scores. SubspaceAD reports state-of-the-art performance across one-shot and few-shot settings without memory banks, auxiliary datasets, prompt tuning, or any form of training.

Overview of SubspaceAD. (Fitting) Patch features from normal images are extracted by a frozen DINOv2 model and a PCA subspace is fitted. (Inference) Test features are projected onto the normal subspace and reconstruction errors yield an anomaly map.
